# webchat

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

- axios
- less

- 该文件需要放在根目录下
- 每次更改配置文件需要停止热部署(ctrl+c),重启服务
